## 🎯 启动步骤

### 1. 启动后端服务
```bash
cd Server_Ai
mvn spring-boot:run
```

**等待后端启动成功**，看到以下日志：
```
Started AiSmartLibraryApplication in X.XXX seconds
```

### 2. 启动前端服务
```bash
cd fronted/web-admin
npm install  # 第一次需要安装依赖
npm run dev
```

**等待前端启动成功**，看到以下输出：
```
  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose
```

### 3. 访问页面
1. 打开浏览器：http://localhost:3000
2. 输入管理员账号密码登录
3. 点击左侧菜单 **"知识库管理"**
4. 看到知识库问答对管理页面

## ❓ 常见问题

### Q1: 菜单点击后没反应？
**A**: 检查：
1. 前端控制台是否有错误
2. 路由是否正确配置
3. 刷新页面后重试

### Q2: 页面空白或报错？
**A**: 检查：
1. 后端服务是否正常运行
2. 浏览器控制台是否有错误
3. 检查 API 接口是否正常（F12 → Network）

### Q3: 导入失败？
**A**: 检查：
1. 文件格式是否正确（.xlsx, .xls, .csv）
2. 第1行是否为标题行
3. 必填字段是否为空
4. 查看导入结果弹窗的错误详情

### Q4: 导出没反应？
**A**: 检查：
1. 浏览器是否阻止了下载
2. 后端服务是否正常
3. 检查浏览器下载设置
