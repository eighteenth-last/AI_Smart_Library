# 知识库导入模板说明

## Excel 导入模板

### 标题行（必须）
第1行必须包含以下列（顺序必须一致）：

| 问题 | 答案 | 分类 | 来源 | 是否公开 |
|------|------|------|------|----------|

## CSV 导入模板

### 标题行（必须）
```csv
question,answer,category,source,is_public
```

### 示例数据
```csv
question,answer,category,source,is_public
图书馆开放时间是什么？,图书馆周一至周日 8:00-22:00 开放，节假日开放时间请关注官网通知。,图书馆业务,system,1
如何借阅图书？,凭借书证在自助借还机上办理借阅手续，每次最多可借5本，借期30天。,图书馆业务,system,1
借书证丢失怎么办？,请携带身份证到图书馆服务台办理挂失和补办手续，补办费用20元。,图书馆业务,admin,1
如何成为签约作者？,您需要发布至少3篇优质作品，且累计阅读量超过10000次，即可申请成为签约作者。,作者问答,admin,1
有哪些经典科幻小说推荐？,"推荐《三体》《银河帝国》《沙丘》等经典科幻作品，这些书籍在本馆均有馆藏。",图书推荐,system,1
```

**注意**：
- CSV 文件必须使用 UTF-8 编码（避免中文乱码）
- 如果内容包含逗号，请使用双引号包裹
- Windows Excel 保存 CSV 时请选择"CSV UTF-8（逗号分隔）"

## 字段值说明

### 是否公开（is_public）
支持以下值（不区分大小写）：
- **公开**: `1`, `公开`, `public`
- **私有**: `0`, `私有`, `private`
- **默认**: 不填写或留空时默认为公开（1）

### 分类（category）
常用分类：
- 图书馆业务
- 作者问答
- 图书推荐
- 其他自定义分类

### 来源（source）
常用来源：
- `system` - 系统录入
- `admin` - 管理员添加
- `author` - 作者回复

## 下载模板

### 方式 1：通过系统导出
1. 进入知识库管理页面
2. 点击"导出数据" → "导出为 Excel"
3. 导出的文件即为标准模板格式
4. 清空数据行，填写新数据即可

### 方式 2：手动创建
1. 新建 Excel 文件
2. 复制上方的标题行
3. 按照示例填写数据
4. 保存为 `.xlsx` 格式

---

## 批量数据建议

### 大批量导入（1000+ 条）
1. 分批导入，每批 500-1000 条
2. 先测试少量数据（10-20条）验证格式
3. 逐步导入完整数据

### 数据整理技巧
1. 使用 Excel 筛选功能去重
2. 使用 Excel 查找替换功能批量修改
3. 使用公式检查字段长度（如 `=LEN(A2)`）
4. 使用数据验证功能限制输入值
